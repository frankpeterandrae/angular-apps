Tow Thin Coats

Ich habe auf dieser Seite, eine Übersicht über Duncan Roades' Tow Thin Coats Farben erstellt.

Diese Seite bietet folgende Funktionen:

  1. Farbsuche: Benutzer können eine Farbeingabe verwenden, um nach bestimmten Farben zu suchen. Die Suchergebnisse werden in einem Raster angezeigt.

  2. Farbraster: Die gefundenen Farben werden in einem Raster dargestellt. Jede Farbe kann angeklickt werden, um weitere Details anzuzeigen.

  3. Farbdetails: Beim Klicken auf eine Farbe im Raster öffnet sich ein Dialog, der detaillierte Informationen über die ausgewählte Farbe anzeigt, einschließlich alternativer Namen und Farbtypen.
